# Todo List App

Aplicaci&oacute;n web de lista de tareas (CRUD) con HTML, CSS y JavaScript vanilla.

## Funcionalidades

- **Crear** tareas con t&iacute;tulo y descripci&oacute;n opcional
- **Leer** tareas ordenadas por fecha (m&aacute;s recientes primero)
- **Modificar** t&iacute;tulo, descripci&oacute;n y estado (completada / no completada)
- **Eliminar** tareas individuales o limpiar todas las completadas

## Tecnolog&iacute;as

- HTML5
- CSS3 (puro, minimalista)
- JavaScript (ES6+, vanilla)

## C&oacute;mo usar

1. Abre `index.html` en tu navegador
2. Escribe un t&iacute;tulo y agrega la tarea
3. Usa los botones para editar, eliminar o marcar como completada

## Estructura

```
├── index.html
├── styles.css
└── script.js
```

## Persistencia

Las tareas se guardan autom&aacute;ticamente en `localStorage` y sobreviven al cerrar o recargar la p&aacute;gina.
